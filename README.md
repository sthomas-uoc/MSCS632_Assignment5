# Code for MSCS 632 Assignment 5
Implement rideshare in Smalltalk and C++ demonstrating Object-Oriented Programming Principles

## Build and run instructions

### C++
The C++ implementation is in the file rideshare_cpp/rideshare.cc

A C++ compiler supporting C++ 11 standards is required.

To compile the application run `g++ -o rideshare_cpp/rideshare rideshare_cpp/rideshare.cc`

To run the application run `./rideshare_cpp/rideshare`

### Smalltalk
The Smalltalk class definitions are in the file rideshare_st/mscs632-a5.st

The worspace demonstrating various OOP features are in the file rideshare_st/mscs632-a5_workspace.st

Import the class definitions and the workspace in Squeak development environment to enable program execution.
